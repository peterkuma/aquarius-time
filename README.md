# Aquarius Time: Python Time Library

**Development status:** in development

Aquarius Time adopts Julian date as the main representation of time.

## Installation

Requirements:

- Python 2.7

Install:

```sh
python setup.py install [--user]
```

## Usage

```python
import aquarius_time as aq

aq.from_datetime(x)
aq.to_datetime(x)
aq.from_iso(x)
aq.to_iso(x)
aq.parse_iso(x)
aq.from_datetime(x)
aq.to_datetime(x)
```
