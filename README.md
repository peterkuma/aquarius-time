# Aquarius Time: Python Time Library

**Development status:** in development

Aquarius Time adopts Julian date as the main representation of time.

## Usage

	import aquarius_time as aq

	aq.from_datetime(x)
	aq.to_datetime(x)
	aq.from_iso(x)
	aq.to_iso(x)
	aq.parse_iso(x)
	aq.from_datetime(x)
	aq.to_datetime(x)
